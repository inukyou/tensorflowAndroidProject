# tensorflowAndroidProject
本程序是一个基于tensorflow的手绘图形识别软件
，目前本程序只能识别：鱼、花、圆圈、猫四种图形

使用了opencv将获取到的图像缩小到28*28并将其灰度化


训练模型和代码https://github.com/inukyou/tensorflow_DrawTrain
